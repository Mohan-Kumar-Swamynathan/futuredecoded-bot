# Background music — no files required

You do **not** need to add any BGM files. The bot creates a subtle ambient track automatically on the first run and reuses it after that:

- `long_form_auto.mp3` — long-form videos
- `shorts_auto.mp3` — Shorts

These are mixed quietly under the narration (about 10–12% volume) so voice stays clear.

## Optional: use your own tracks later

If you ever want custom royalty-free music, drop files here:

- `long_form.mp3`
- `shorts.mp3`

Custom files override the auto-generated ones. Until then, do nothing — it works out of the box.
